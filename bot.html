WEB BOT DETECTION USING USER AGENT STRING

  • Super user
  • August 28, 2020
  • Hits:145

We can easily detect bot using the user agent string and the client public ip address. IAB releases a whitelist of known browers which is updated every month. We pull this list generate a TRIE and then match the browsers obtained from the user agent in it. Similarly IAB also releases a Blacklist of known crawlers and bots. We can implement the blacklist for the same.

Besides the user agent string , client IP can also be used for detecting bot. Match the user public ip against the data center public ips published by all major providers. For optimisation match the user ip directly on the CIDR format data center allocated Public IPs. This can be done easily using the prefix matching capability of Trie.
